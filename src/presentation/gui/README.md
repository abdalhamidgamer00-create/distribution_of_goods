# واجهة Streamlit - نظام توزيع البضائع

## التشغيل

### الطريقة 1: استخدام run_gui.py
```bash
python run_gui.py
```

### الطريقة 2: استخدام main.py
```bash
python main.py --gui
```

### الطريقة 3: مباشرة مع Streamlit
```bash
streamlit run src/app/gui/app.py
```

## الصفحات

1. **الرئيسية**: تشغيل جميع الخطوات
2. **ملفات التحويل**: عرض وتحميل ملفات التحويل بين الفروع
3. **الفائض المتبقي**: عرض وتحميل ملفات الفائض المتبقي
4. **النقص**: عرض وتحميل ملفات النقص

## الميزات

- واجهة عربية كاملة
- عرض البيانات في جداول
- تحميل فردي وجماعي
- فلترة حسب الفرع والفئة
- تشغيل الخطوات من الواجهة
